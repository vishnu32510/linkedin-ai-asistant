Generate personalized LinkedIn notes and messages using AI

1 Get Your OpenAI API Key

You'll need your own OpenAI API key to use this extension:

  1. Visit OpenAI Platform
  2. Sign up or log in to your account
  3. Go to API Keys section
  4. Create a new API key (starts with "sk-proj-" or "sk-")

Note: OpenAI charges based on usage. Check their pricing page for details.

2 Set Up Your API Key

Add your API key to the extension:

  1. Right-click the extension icon in your toolbar
  2. Click "Options"
  3. Paste your OpenAI API key
  4. Click "Save"
  5. (Optional) Click "Test Connection" to verify
3 Start Using on LinkedIn

Navigate to any LinkedIn profile page:

  1. Go to linkedin.com/in/username
  2. You'll see two buttons:
    • "Generate LinkedIn Note" - For connection requests
    • "Generate LinkedIn Message" - For direct messages
  3. Click either button and optionally add a job description
  4. The extension will automatically generate and paste the content!
